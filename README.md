\# 🎮 Guess the Number



\## 📌 Cel projektu

Prosta gra w Pythonie, w której program losuje liczbę, a gracz musi ją odgadnąć. Projekt został stworzony do nauki podstaw programowania.



\## ▶️ Jak grać

\- Program losuje liczbę od 1 do 100  

\- Wpisujesz swoje liczby w konsoli  

\- Otrzymujesz podpowiedzi: „za mało” lub „za dużo”  

\- Wygrywasz, gdy zgadniesz liczbę  



\## 🚀 Jak uruchomić

1\. Zainstaluj Python 

2\. Pobierz plik `guessthenumber.py`  

3\. Uruchom w terminalu python guessthenumber.py
